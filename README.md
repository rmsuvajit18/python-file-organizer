# File Organizer Automation Tool

## 📌 Project Overview
The File Organizer Automation Tool is a Python automation project that organizes files in a selected folder based on their file type.

The script scans all files in a folder, detects their file extensions, and automatically moves them into categorized folders such as Images, Documents, Videos, Music, and Others.

This project demonstrates Python concepts like file handling, automation, and working with built-in libraries.

---

## 🚀 Features
- Automatically scans files in a folder
- Detects file extensions
- Creates folders automatically
- Moves files into correct categories
- Supports multiple file types
- Handles unknown file types using an **Others** folder

---

## 🛠 Technologies Used
- Python
- OS Module
- Shutil Module
- Command Prompt / Terminal
